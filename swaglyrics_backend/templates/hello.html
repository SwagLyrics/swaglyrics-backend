SwagLyrics Backend

The SwagLyrics Backend and API is housed here.
A key function of the backend is using a database and the Genius API to support songs that the main application cannot fetch lyrics for by itself. If that is not possible by even the backend (for example, if lyrics don't exist on Genius), it avoids redundancy by adding unsupported songs to the list below as well as opening GitHub Issues for these songs for rapid iteration and review.
The backend also checks these songs with the Spotify API and confirms the validity of data it is provided.
